# Foreign Trade Automation

This is my first GitHub project for learning Python, Codex, and B2B foreign trade automation.

## Project Goal

The goal of this project is to build simple automation tools for foreign trade work, especially for Made-in-China product operation, product data organization, keyword research, FAQ generation, and SEO content preparation.

## Folder Structure

* `data/`
  Store original business data, such as product information, keyword lists, FAQ questions, and Made-in-China product data.

* `scripts/`
  Store Python scripts for automation tasks, such as cleaning product data, generating product titles, and organizing keywords.

* `outputs/`
  Store results generated by Python scripts, such as cleaned Excel files, optimized titles, FAQ outputs, and SEO keyword results.

* `prompts/`
  Store useful prompts for Codex, ChatGPT, and automation task planning.

## Current Automation Script

### Made-in-China Product Title Generator

This script reads product data from:

```text
data/sample_products.csv
```

It generates product title drafts and saves the result to:

```text
outputs/generated_titles.csv
```

To run the script locally, use:

```powershell
python scripts/generate_titles.py
```

This is my first simple automation practice for B2B foreign trade product operation.
